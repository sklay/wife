jQuery 自述

程序员行业有一句名言： “偷懒是程序员的美德”。

“偷懒”这个词在生活中，意味着坏的习惯，贬义词，失败的人， 但是在软件编程当中，意味着提高效率，降低成本。

我们程序员开发众多的项目和产品，提高了企业，家庭，学校等人们的信息分析和获取的效率。 程序员设计了众多的算法提高了计算机cpu和内存的使用效率， 过去的程序员前辈们也为了现在和将来的程序员 封装了很多的类库和框架， 让我们提高开发程序的效率。

jQuery是免费的（MIT许可证），流行的JavaScript库， 2006年由John Resig 创建， 设计目的是 简化浏览器的JavaScript脚本编程。 （本质上，作者还是为了偷懒，哈哈）

JQuery的口号是： write less ，do more 。（写更少的代码，做更多的功能， 这么郎朗上口的口号， 果然是 懒人不可怕，就怕懒人有文化，啪啪啪）

jQuery 官方说法

jquery是一个快速，小巧，功能丰富的JavaScript库。

jquery API使得跨浏览器的HTML文档操作，事件处理，动画，ajax开发更加方便，简单。

其良好的通用性和可扩展性，jquery改变了数以百万开发者编写JavaScript的方式。

所有的软件编程说白了，都是对数据的读写（增删改查），无论你学习什么语言或者框架， 当你能够使用它对相关数据进行熟练读写时，你已经算掌握了他的基本功能啦。

jquery最核心的内容就是 对html页面当中的元素进行查询，然后获取其属性值，修改其属性值。（俗称： DOM操作）。

jQuery 优势归纳

1、轻量级。jQuery非常轻巧，采用UglifyJS(https://github.com/mishoo/UglifyJS)压缩后，大小保持在30KB左右。

2、强大的选择器。

  • jQuery 允许开发者使用从CSS1到CSS3几乎所有的选择器。
  • jQuery独创的高级而复杂的选择器。
  • 还可以加入插件使其支持Xpath选择器。
  • 甚至开发者可以编写属于自己的选择器。

3、出色的DOM操作的封装。

4、可靠的事件处理机制

5、完善的ajax。jQuery将所有的Ajax操作封装到一个函数$.ajax()里,并且解决复杂的浏览器兼容性和XMLHttpRquest对象的创建和使用的问题。

6、不污染顶级变量。jQuery只建 立一个名为jQuery的对象，其所有的函数方法都在这个对象之下。该特性使jQuery可以与 其他JavaScript库共存。

7、出色的浏览器兼容性。

8、链式操作方式。

9、隐式迭代。当用jQuery找到带有".myclass"类的全部元素，然后隐藏它们时，无需循环遍历每一个返回的元素。相反，jQuery里的方法都被设计成自动操作对象集合，而不是单独的对象，这使得大量的循环结构变得不再必要，从而大幅度地减少代码。

10.行为层与结构层的分离。

11.丰富的插件支持。

12.完善的文档。

13.开源。

看到这儿，相信您对jQuery在概念上也已经有了大概的认识

jQuery 基础示例

下面的例子演示了 jQuery 的 hide() 函数，隐藏了 HTML 文档中所有的 <p> 元素。

实例 

<html>
<head>
<script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript">
$(document).ready(function(){
$("button").click(function(){
$("p").hide();
});
});
</script>
</head>
<body>
<h2>This is a heading</h2>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
<button type="button">Click me</button>
</body>
</html>
