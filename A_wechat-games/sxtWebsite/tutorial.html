模板

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">//如果使用IE浏览器，用"edge"进行渲染，即IE11
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <!-- 上述3个meta标签*必须*放在最前面，任何其他内容都*必须*跟随其后！ -->
    <title>Bootstrap 101 Template</title>

    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet">

    <!-- HTML5 shim and Respond.js for IE8 support of HTML5 elements and media queries -->
    <!-- WARNING: Respond.js doesn't work if you view the page via file:// -->
    <!--[if lt IE 9]>
 <script src="https://cdn.bootcss.com/html5shiv/3.7.3/html5shiv.min.js"></script>//让浏览器可以识别Html5的新标签
 <script src="https://cdn.bootcss.com/respond.js/1.4.2/respond.min.js"></script>//让低版本浏览器可以使用CSS3的媒体查询
 <![endif]-->
</head>
<body>
<h1>你好，世界！</h1>

<!-- jQuery (necessary for Bootstrap's JavaScript plugins) -->
<script src="https://cdn.bootcss.com/jquery/1.12.4/jquery.min.js"></script>
<!-- Include all compiled plugins (below), or include individual files as needed -->
<script src="js/bootstrap.min.js"></script>
</body>
</html>

简单方便将文字对齐的类，一般应用在p标签（用在div上不规范，用在span标签则不生效）

<div class="container">
    <p class="text-left">文字左对齐</p>
    <p class="text-center">文字居中对齐</p>
    <p class="text-right">文字右对齐</p>
</div>

强调：这些是强调类，通过颜色来表示强调（p元素添加了这些类 ）

<div class="container">
    <p class="text-muted">muted减弱效果</p>
    <p class="text-warning">text-warning警告效果</p>
    <p class="text-danger">text-danger错误提示</p>
    <p class="text-info">text-info信息</p>
    <p class="text-success">text-success这是正确的强调信息</p>
</div>

表单：单独的form标签会被自动赋予一些全局样式。所有在设置了.form-control类里的<input>、<textarea>和<select>元素都被默认设置宽度为100%，将label和input、textarea、select这些元素一起包裹在.form-group中可以获得最好的排列

<div class="container">
    <div class="row">
        <div class="col-md-12">>
                <div class="form-group">
                    <label for="user" class="control-label col-md-2">用户名：</label> //control-label文字居中
                    <div class="col-md-10">
                        <input type="text" id="user" class="form-control" disabled/>//.input-lg、.input-sm控制输入框尺寸
                    </div>
                </div>
                <div class="form-group">
                    <select name="" id="" class="form-control">
                        <option value="1">江苏</option>
                        <option value="2">北京</option>
                        <option value="3">浙江</option>
                        <option value="4">云南</option>
                    </select>
                </div>
            </form>
        </div>
    </div>
</div>

导航：可为导航添加.navbar-fixed-top或.navbar-fixed-bottom将导航固定在顶部或者底部。

<!--navbar声明导航样式，navbar-default声明可以反色-->
<nav class="navbar navbar-default">
    <!-- container-fluid的width是100%,而container是用媒体查询获得动态的尺寸-->
    <div class="container-fluid">
        <!--navbar-header元素向左浮动-->
        <div class="navbar-header">
            <!-- navbar-brand展示了一个Logo-->
            <a href="#" class="navbar-brand">
                <!-- 图片注意尺寸，导航栏默认高度为50px,图片超出尺寸，需重新布局-->
                <img src="img/btn.gif" alt=""/>
            </a>
        </div>
        <!-- nav是专门为导航中的列表加样式，navbar是为整个导航添加样式，navbar-nav是让li左浮动--->
        <ul class="nav navbar-nav">
            <!-- active为当前活动的元素-->
            <li class="active"><a href="#">首页</a></li>
            <li><a href="#">业务介绍</a></li>
            <li><a href="#">产品中心</a></li>
            <li><a href="#">联系我们</a></li>
            <li><a href="#">成功案例</a></li>
            <li class="dropdown">
                <a class="dropdown-toggle" data-toggle="dropdown">
                    chose your country
                    <span class="caret"></span>
                </a>
                <ul class="dropdown-menu">
                    <li><a href="#">China</a></li>
                    <li><a href="#">America</a></li>
                    <li><a href="#">France</a></li>
                    <li><a href="#">Italy</a></li>
                    <li class="divider"></li>
                    <li><a href="#">London</a></li>
                    <li><a href="#">...</a></li>
                </ul>
            </li>

        </ul>
//在导航栏中添加搜索框
        <form action="" class="navbar-form navbar-right">
            <a href="#" class="navbar-link">登录</a>
            <a href="#" class="navbar-link">注册</a>
            <input type="text" class="form-control" placeholder="aaa"/>
            <button class="btn btn-primary">搜索</button>
        </form>

    </div>

</nav>
